Nosso objetivo

Se você acessou nosso "Sobre", provavelmente já sabe do que irei falar. Caso não tenha acessado ainda, Clique aqui

No entanto, se estiver com pressa e quiser ir direto ao ponto, acho interessante que conheça rapidamente a motivação para a criação desse site.

Pensando em ajudar aqueles que estão procurando formas de estudar de forma gratuita, reuni uma série de sites que julguei muito bons para estudar, seja para o enem, para o seu colégio, para capacitação ou apenas por curiosiidade. Espero que goste!

Enem

Se você, assim como eu, está começando (ou já está nessa faz tempo) a se preparar para o Enem, tenho algumas dicas que podem ser muito valiosas. Quando se trata do Exame nacional, não podemos perder tempo. Pensando nisso, listei uma série de sites que posuem ótimo conteúdo.

Para ser direcionado ao site basta clicar no nome do mesmo.

Proenem

O Proenem possui uma plataforma paga, mas também posta muitas aulas no seu canal no YouTube. As aulas são muito boas e vale muito a pena dar uma olhada!

Planejativo

O planejativo é um site que possibilita a criação de um cronograma super organizado de estudos. O próprio site recomenda vídeos para cada tema e possui muitas questões para serem resolvidas

Projeto Medicina

Se você, assim como eu, passa horas procurando exercícios na internet, seus problemas acabaram! Nesse site é possível encontrar as mais diversas questões sobre os mais diversos conteúdos.

Curso Enem gratuito

Este é um outro site muito bom. Com ele, é possível assistir a aulas e responder questões do Enem de forma muito simples.

Inep

No site do Inep é possível encontrar as provas com gabarito do Enem dos anos anteriores. Tentar resolver as questões e ver resoluções comentadas (basta pesquisar no YoTube) é uma ótima forma de aprender a lidar com a prova.

Cursos

Se você quer aprimorar seus conhecimentos, explorar possíveis áreas de atuação ou aprender algo novo por pura curiosidade, aqui vão alguns sites onde será possível realizar tudo o que mencionei.

Para ser direcionado ao site basta clicar no nome do mesmo.

Coursera

Este site possui cursos de todo o mundo, e a maioria possui legendas e emite certificados de forma gratuita. Nele, é possível encontrar uma vasta quantidade de cursos das mais diversas áreas possíveis.

Rocketseat

Se você sonha em aprender desenvolvimento web, a rocketseat é um ótimo site

Duolingo

O duolingo te ensina diversas línguas de uma forma dinâmica e voltada para o cotidiano. Para os que sonham em aprender outros idiomas, é uma ótima ferramenta

EDX

Para os que possuem um nível de inglês bom, esse site fornece cursos gratuitos de universidades como Harvard, Berkeley e MIT de forma gratuita.

Code iot

Esse site possui 6 cursos que envolvem Internet das coisas, Arduino e Linguagem de programação. Para aqueles que se interessam pelo meio, vale muito a pena, uma vez que os professores explicam muito bem e o conteúdo é muito bom.
